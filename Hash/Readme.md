## _Hash_

O _Hash_ é uma coleção de dados estruturado, que trabalha com chave valor.

### Estrutura
__Declaração__:

    k = {key:'name key', value: 'number' }

                    ou

    k = {key:'name key',
         value: 'number age' }

__Exemplo__:

    k = {nome: "João", age: 26}


__Acessar os itens do__ _Hash_:

    k.nome
    k.age
