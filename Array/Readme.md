## Array em _JavaScript_

### Declaração de um Array

    nroPrimos = [1, 2, 3, 5, 7, 11]

#### Retirar Último Elementos no Array

    nroPrimos.pop()

#### Inserir um elemento no Array

    nroPrimos.push(13)
              ou
    nroPrimos[6] = 17

#### Saber o tamanho do Array

    nroPrimos.length
